About the Project

This MR Productivity App focuses on giving users accurate, real‑world utilities inside mixed reality. We begin by building a measuring tape you can stretch between two points, then add measurement labels that update in inches and centimeters.

Next, we incorporate a level tool to check whether surfaces or frames are properly aligned.

MR Measuring Tape

Place starting and ending points, dynamically calculate distance, position labels, and maintain visibility as you measure.

Leveler

Place a leveling bar and rotate it to check if a surface or line is level.

Milestones

  1. Configure Unity project + XR All‑in‑One SDK
  2. Implement MR measuring tape with start/end points
  3. Add measurement labels (inches + cm)
  4. Build level tool + surface alignment
  5. Implement reference frame mount generator
  6. Project polish and MR performance tuning
